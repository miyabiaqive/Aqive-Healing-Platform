# Aqive v2.0 - 頁面導航地圖

## 📱 完整頁面結構

```
index.html (iPhone 容器)
│
├─ home.html ..................... 首頁
│   ├─ mood-checkin.html ......... 情緒 check-in
│   ├─ player.html ............... 音頻播放器
│   ├─ ai-chat.html .............. AI 對話
│   ├─ breath.html ............... 呼吸練習
│   └─ settings.html ............. 設置
│
├─ discover.html ................. 探索課程
│   └─ course-detail.html ........ 課程詳情
│       └─ player.html ........... 開始學習
│
├─ ai-chat.html .................. AI 陪伴
│   ├─ breath.html ............... 快速呼吸練習
│   └─ player.html ............... 推薦音頻
│
└─ profile.html .................. 個人檔案
    ├─ course-detail.html ........ 我的課程
    └─ settings.html ............. 設置
```

---

## 🎯 核心頁面詳解

### 1️⃣ index.html - 主入口
**功能**：iPhone 15 Pro 容器  
**特色**：
- 393 x 852 px 比例
- 劉海設計
- 深藍紫邊框
- iframe 嵌套系統

**入口頁面**：home.html

---

### 2️⃣ home.html - 首頁
**功能**：主儀表板  
**元素**：
- 個人問候（Good afternoon, Jessica）
- 情緒 check-in 快速入口
- Recently played（2個卡片）
- Your favorites（列表式）
- 底部導航（4個 Tab）

**可導航至**：
- mood-checkin.html（點擊情緒 check-in）
- player.html（點擊任何音頻卡片）
- settings.html（點擊右上角設置圖標）
- discover.html（底部導航）
- ai-chat.html（底部導航）
- profile.html（底部導航）

---

### 3️⃣ mood-checkin.html - 情緒記錄
**功能**：情緒追蹤與記錄  
**元素**：
- 當前情緒顯示（Today mood: happiness）
- 週曆視圖（Week of Aug, 3）
- 情緒歷史點擊（View history）
- Recommended for you（4個推薦卡片）

**可導航至**：
- home.html（點擊左上角返回或關閉）
- player.html（點擊推薦內容）

**視覺特色**：
- 山景漸層背景
- 玻璃擬態卡片
- Emoji 情緒圖標

---

### 4️⃣ discover.html - 探索
**功能**：課程探索與搜尋  
**元素**：
- 搜尋框
- 分類篩選（All / Sleep / Anxiety / Stress / Focus）
- Featured 課程（大卡片）
- Popular courses（列表式）
- 底部導航

**可導航至**：
- course-detail.html（點擊任何課程）
- home.html（底部導航）
- ai-chat.html（底部導航）
- profile.html（底部導航）

**視覺特色**：
- 白色背景
- 評分星星顯示
- 漸層特色區塊

---

### 5️⃣ ai-chat.html - AI 對話
**功能**：AI 陪伴與對話  
**元素**：
- AI 頭像與在線狀態
- 對話訊息（AI 與用戶）
- 快速行動卡片（呼吸練習 / 音樂）
- 快速話題按鈕
- 訊息輸入框
- 底部導航

**可導航至**：
- breath.html（點擊呼吸練習卡片）
- player.html（點擊音樂卡片）
- home.html（底部導航）
- discover.html（底部導航）
- profile.html（底部導航）

**視覺特色**：
- 對話泡泡（灰色 AI / 紫藍漸層用戶）
- 時間戳記
- 快速行動卡片

---

### 6️⃣ player.html - 播放器
**功能**：音頻播放  
**元素**：
- 封面圖片（大）
- 曲目標題與作者
- 進度條（時間顯示）
- 播放控制（上一首 / 播放暫停 / 下一首）
- 收藏按鈕
- 播放列表按鈕

**可導航至**：
- home.html（點擊下拉按鈕）

**視覺特色**：
- 山景漸層背景
- 全螢幕沉浸式
- 大圓角封面

---

### 7️⃣ profile.html - 個人檔案
**功能**：用戶資料與進度  
**元素**：
- 頭像與名稱
- 統計數據（Days streak / Sessions / Hours）
- Energy Level（3個進度條）
- My Courses（進度顯示）
- 底部導航

**可導航至**：
- settings.html（點擊右上角設置）
- course-detail.html（點擊課程）
- home.html（底部導航）
- discover.html（底部導航）
- ai-chat.html（底部導航）

**視覺特色**：
- 紫藍漸層頂部
- 白色內容區
- 進度條視覺化

---

### 8️⃣ breath.html - 呼吸練習
**功能**：引導呼吸練習  
**元素**：
- 呼吸動畫圓圈（3層）
- 呼吸指示文字（Breathe In / Hold / Breathe Out）
- 輪次進度（Round 1 of 5）
- 進度點（5個）
- 開始/停止按鈕

**可導航至**：
- home.html（點擊關閉）

**視覺特色**：
- 藍紫粉漸層背景
- 呼吸動畫（8秒循環）
- 沉浸式全螢幕

**互動邏輯**：
- 點擊 Start → 開始動畫
- 文字每 2 秒切換（Breathe In → Hold → Breathe Out → Hold）
- 點擊 Stop → 停止動畫

---

### 9️⃣ course-detail.html - 課程詳情
**功能**：課程介紹與學習  
**元素**：
- 課程封面圖（Hero）
- 課程標題與教師資訊
- 評分與學生數
- 課程簡介
- Session 列表（鎖定狀態）
- 底部 CTA 按鈕（Start Learning）

**可導航至**：
- discover.html（點擊左上角返回）
- player.html（點擊 Session 或 Start Learning）

**視覺特色**：
- Hero 圖片 + 漸層遮罩
- 教師頭像
- Session 鎖定圖示

---

### 🔟 settings.html - 設置
**功能**：應用設置  
**元素**：
- Account 群組（Edit Profile / Privacy）
- Preferences 群組（Notifications / Dark Mode / Language）
- Support 群組（Help Center / Contact Us）
- Log Out 按鈕

**可導航至**：
- profile.html（點擊左上角返回）

**視覺特色**：
- 白色背景
- 分組標題（灰色小字）
- 列表式設置項

## 💡 快速測試指南

### 測試情緒記錄功能
1. 開啟 index.html
2. 點擊 "How are you feeling?" 卡片
3. 查看週曆與推薦內容
4. 點擊任一推薦 → 進入播放器

### 測試 AI 對話功能
1. 點擊底部導航的 "AI Chat"
2. 查看對話歷史
3. 點擊 "Breathing Exercise" 卡片
4. 體驗呼吸練習動畫

### 測試課程探索
1. 點擊底部導航的 "Discover"
2. 瀏覽 Popular courses
3. 點擊任一課程 → 查看詳情
4. 點擊 "Start Learning" → 進入播放器

### 測試個人檔案
1. 點擊底部導航的 "Profile"
2. 查看統計數據與能量進度
3. 點擊 "My Courses" 中的課程
4. 點擊右上角設置圖標 → 查看設置
