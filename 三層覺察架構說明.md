# 三層覺察架構 - 完整實現文檔

## 🎯 設計目標

兼顧三個核心層面：
1. **理性可接受度**（邏輯化的語言）
2. **潛意識觸發**（讓內在被看見）
3. **可導向行動的分類**（便於智能推薦）

---

## 📊 三層覺察架構

### 層級設計理念

| 層級 | 目的 | 心理機制 | 語言特色 |
|------|------|----------|----------|
| **層一：現象覺察** | 從模糊轉為具體觀察 | 降低心理防衛 | 具體、身體化 |
| **層二：情境覺察** | 識別外在壓力源 | 外化問題 | 客觀、情境化 |
| **層三：根源覺察** | 觸碰內在狀態 | 潛意識浮現 | 感受化、深層 |
| **層四：放鬆意象** | 探索深層需求 | 意象投射 | 開放、直覺 |
| **層五：療癒意圖** | 明確行動期待 | 目標設定 | 行動導向 |

## 🧩 五種心理能量型

### 1. 頭腦過載型（Mental Overload）

**特徵**：
- 思緒不斷、無法放鬆
- 過度理性、情緒麻木
- 頭痛肩頸緊繃
- 需要「關掉手機」才能放鬆

**觸發選項**：
```
Q1: 頭痛肩頸緊繃 / 身體還好但心很累
Q2: 工作目標壓力
Q3: 無法放鬆，腦袋停不下來
Q4: 關掉手機／斷開連結
Q5: 提醒我還在呼吸
```

**推薦邏輯**：
- **立即練習**：撓境身體覺察（6分鐘回到身體）
- **閱讀文章**：《過度理性是情緒麻木的表現》
- **深度課程**：身心放鬆深度課程（4週）

**AI 洞見**：
> "過度理性常常是情緒麻木的表現。當我們習慣用思考來控制一切，身體的訊號就被忽略了。"

---

### 2. 神經亢奮型（Nervous Arousal）

**特徵**：
- 焦慮、失眠
- 胸悶呼吸急促
- 神經系統長期警戒
- 對未來焦慮

**觸發選項**：
```
Q1: 睡不好／失眠 / 胸悶呼吸急促
Q3: 對未來焦慮，缺乏安全感
Q4: 沒辦法放鬆
Q5: 讓我靜下來
```

**推薦邏輯**：
- **立即練習**：安撫神經系統呼吸法（5分鐘）
- **閱讀文章**：《你的焦慮，其實是身體在說話》
- **深度課程**：深度睡眠修復計畫（21天）

**AI 洞見**：
> "焦慮不是你的問題，是神經系統需要被安撫。它一直在保護你，現在需要告訴它：你是安全的。"

---

### 3. 能量耗竭型（Energy Depleted）

**特徵**：
- 疲倦沒動力
- 空虛、失去熱情
- 想睡覺休息
- 想重新感覺到自己

**觸發選項**：
```
Q1: 疲倦沒動力
Q3: 空虛，不知道為何而做
Q4: 睡覺／休息
Q5: 讓我重新感覺到自己
```

**推薦邏輯**：
- **立即練習**：能量補充冥想（10分鐘）
- **閱讀文章**：《允許自己休息，不是軟弱》
- **深度課程**：自我滋養7日計畫

**AI 洞見**：
> "空虛和疲倦，是靈魂在提醒你：該回來照顧自己了。你已經很久沒有為自己做些什麼了。"

---

### 4. 外部糾結型（External Entanglement）

**特徵**：
- 被期待綁住
- 家庭關係緊張
- 人際溝通卡住
- 想旅行換環境
- 需要建立界限

**觸發選項**：
```
Q2: 家庭關係緊張 / 人際溝通卡住
Q3: 被期待綁住，喘不過氣
Q4: 旅行／換個環境
Q5: 幫我建立界限
```

**推薦邏輯**：
- **立即練習**：界限覺察練習（8分鐘）
- **閱讀文章**：《說不，是一種自我照顧》
- **深度課程**：關係中的自我覺察（6週）

**AI 洞見**：
> "你不需要為所有人負責。建立健康的界限，不是自私，是自愛。"

---

### 5. 內在失焦型（Inner Unfocused）

**特徵**：
- 方向迷惘
- 自我懷疑
- 想重新開始但不知從何下手
- 需要釐清方向

**觸發選項**：
```
Q2: 自我懷疑／方向迷惘
Q3: 想重新開始但不知從何下手
Q5: 幫我釐清方向
```

**推薦邏輯**：
- **立即練習**：內在指引冥想（12分鐘）
- **閱讀文章**：《迷惘是找到自己的必經之路》
- **深度課程**：生命藍圖覺察工作坊（8週）

**AI 洞見**：
> "迷惘不是失敗，是靈魂在邀請你停下來，重新聆聽內在的聲音。答案一直都在你心裡。"

## 💬 對話式問題設計

### Q1：現象覺察（身體訊號）

**問題**：
> "這幾天，你的身體最常出現哪種訊號？"

**副標題**：
> "選擇最貼近你感受的選項"

**選項設計**：
- 😴 睡不好／失眠
- 😫 頭痛肩頸緊繃
- 😮‍💨 胸悶呼吸急促
- 😔 疲倦沒動力
- 🤔 身體還好，但心很累

**設計理念**：
- 具體身體感受（降低抽象性）
- 涵蓋常見症狀
- 最後一項允許「身心分離」的狀態

---

### Q2：情境覺察（壓力來源）

**問題**：
> "這段時間，讓你最焦慮的情境是哪一類？"

**副標題**：
> "誠實地面對自己的感受"

**選項設計**：
- 🎯 工作目標壓力
- 👨‍👩‍👧 家庭關係緊張
- 💬 人際溝通卡住
- ❓ 自我懷疑／方向迷惘
- 😌 目前壓力還好

**設計理念**：
- 外化問題（不是「你的問題」，是「情境」）
- 涵蓋主要生活領域
- 允許「沒有壓力」的回答

---

### Q3：根源覺察（內在狀態）

**問題**：
> "當你獨處時，最常浮現的感受是？"

**副標題**：
> "讓潛意識的聲音被聽見"

**選項設計**：
- 😰 無法放鬆，腦袋停不下來
- 😤 被期待綁住，喘不過氣
- 😶 空虛，不知道為何而做
- 😟 對未來焦慮，缺乏安全感
- 🌱 想重新開始但不知從何下手

**設計理念**：
- 深入內在（非外在表現）
- 「獨處時」降低社會面具
- 觸碰真實感受

---

### Q4：放鬆意象（深層需求）

**問題**：
> "當你想到「放鬆」時，腦中第一個畫面是？"

**副標題**：
> "跟隨直覺選擇"

**選項設計**：
- 😴 睡覺／休息
- ✈️ 旅行／換個環境
- 📵 關掉手機／斷開連結
- 🧘 靜坐／冥想
- 😞 沒辦法放鬆

**設計理念**：
- 意象投射（繞過理性防衛）
- 揭示潛意識需求
- 「第一個畫面」強調直覺

---

### Q5：療癒意圖（行動期待）

**問題**：
> "如果有一個聲音能幫你回來，你希望它幫你做什麼？"

**副標題**：
> "這是你內在最深的渴望"

**選項設計**：
- 🤫 讓我靜下來
- 🧭 幫我釐清方向
- 🌬️ 提醒我還在呼吸
- 💗 讓我重新感覺到自己
- 🛡️ 幫我建立界限

**設計理念**：
- 行動導向（直接連結推薦）
- 深層渴望（不是表面需求）
- 賦權語言（「幫你」而非「教你」）

## 🔄 智能推薦邏輯

### 推薦層次設計

```
立即行動（3-10分鐘）
    ↓
深入了解（3-5分鐘閱讀）
    ↓
系統學習（數週課程）
```

### 為什麼要三層次？

1. **降低門檻**：不是每個人都準備好上課
2. **漸進引導**：從體驗 → 理解 → 學習
3. **提高轉化**：給不同準備度的用戶選擇

## 🎨 視覺設計原則

### 問題頁面（quiz.html）

- **進度條**：視覺化進度（1/5 → 5/5）
- **層級標籤**：明確顯示當前層級
- **Blob 動畫**：柔和、有機的視覺元素
- **即時反饋**：選中項目有邊框高亮

### 推薦頁面（recommendations.html）

- **AI 分析卡片**：漸層背景 + 玻璃擬態
- **推薦理由框**：淺色邊框 + icon
- **層次分明**：最推薦 > 閱讀 > 深度課程
- **行動按鈕**：大、明顯、有視覺吸引力
