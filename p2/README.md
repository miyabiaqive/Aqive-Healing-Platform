## 🎯 首次用戶 Onboarding 流程

### 問題覺察 → 智慧推薦

當用戶首次訪問平台時，會自動進入科學化的壓力評估流程：

#### 1️⃣ 歡迎引導（onboarding-zh.html）
- **品牌願景傳達**：「在你忙碌的日常裡，這 10 分鐘是回到自己、傾聽身體的機會」
- **功能介紹**：科學化評估 + 智慧推薦 + 持續追蹤
- **選項**：開始評估 / 暫時跳過

#### 2️⃣ PSS-10 壓力評估（pss-assessment-zh.html）
- **基於 PSS-10 量表**：10 題標準化壓力感知問題
- **語言優化**：針對高階白領創業者語境調整
- **補充評估**：
  - 身體訊號（2題）：呼吸急促、胸悶、肌肉緊繃、疲憊感
  - 心智狀態（1題）：工作停不下來、腦中持續思考
- **互動體驗**：
  - 進度條實時顯示
  - 選項點擊後自動前進
  - 支持返回修改答案

#### 3️⃣ 結果分析與推薦（assessment-result-zh.html）
根據分數自動分類並推薦對應內容：

**🟢 壓力可控型（PSS 0-13分）**
- 狀態：能有效管理日常壓力
- 推薦：基礎能量穩定課程、日常冥想、晨間能量喚醒

**🟡 壓力中等偏高（PSS 14-26分）**
- 狀態：正經歷中高程度壓力
- 推薦：深度呼吸引導、身體掃描放鬆、壓力管理工作坊、AI 陪伴對話

**🔴 能量耗竭型（PSS 27-40分）**
- 狀態：高度壓力伴隨身體訊號
- 推薦：倦怠修復療程、睡眠修復引導、邊界設定工作坊、緊急放鬆練習、AI 情緒支援
- 建議：考慮尋求專業協助

### 數據儲存與追蹤

- **LocalStorage 儲存**：評估結果、時間戳記、詳細答案
- **個人檔案展示**：在「我的」頁面顯示最近評估分數和日期
- **定期重測**：建議每月重新評估，追蹤壓力變化趨勢
- **智慧推薦**：根據歷史數據優化內容推薦

### 品牌語言整合

- **引導語**：「在你忙碌的日常裡，這 10 分鐘是回到自己、傾聽身體的機會」
- **結果頁願景**：「找回內在能量，活出真我」
- **陪伴感**：「讓我們陪伴你，一起踏上這段能量回穩的旅程」

### 建議流程

### 設計 Token

#### 間距系統
```
4px  = 1 unit
8px  = 2 units
12px = 3 units
16px = 4 units
24px = 6 units
32px = 8 units

#### 陰影系統

md: 0 4px 6px rgba(0,0,0,0.1)
lg: 0 10px 15px rgba(0,0,0,0.1)
xl: 0 20px 25px rgba(0,0,0,0.15)
```
