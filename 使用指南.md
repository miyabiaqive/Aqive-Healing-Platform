## 🚀 三種開啟方式

### 方式 1：完整體驗（推薦）⭐
```bash
1. 雙擊打開 index.html
2. 在 iPhone 容器中完整瀏覽
3. 點擊各個導航自由探索
```

### 方式 2：本地伺服器（最佳）
```bash
# 使用 Python
cd "Aqive app"
python3 -m http.server 8000

# 瀏覽器訪問
http://localhost:8000/index.html
```

### 方式 3：單頁預覽
```bash
直接打開任何 .html 文件
例如：home.html, discover.html 等
```

```
index.html → home.html
    ↓ 點擊「晨間冥想」

### 💭 情緒陪伴場景
```

    ↓ 點擊「AI 陪伴」

    → player.html (冥想引導)

### 🌬️ 快速放鬆場景
```
home.html
    ↓ 點擊「呼吸練習」
    → breath.html (2分鐘呼吸法)
    ↓ 完成
    ← 返回首頁

### ✨ 能量評估場景
```
home.html
    ↓ 點擊「能量測驗」
    → quiz.html (問卷)

### 📚 課程學習場景
```
home.html 或 discover.html
    ↓ 點擊任一課程
    → course-detail.html
    ↓ 點擊「開始學習」
    → player.html
```

### ⚡ 產品使用場景
```
home.html 或 profile.html
    ↓ 點擊「我的產品」
    → product.html
    ↓ 點擊「啟動」
    → product-activate.html (能量同步儀式)
```

## 🔧 常見問題

### Q1: 打開 index.html 沒有顯示內容？
**A:** 確保 home.html 在同一目錄下。或直接打開 home.html 單頁預覽。

### Q2: 圖標沒有顯示？
**A:** 需要網絡連接來載入 Font Awesome CDN。或下載離線版本。

### Q3: 樣式跑掉了？
**A:** 確保 styles.css 在同一目錄下。檢查 Tailwind CDN 連接。

### Q4: 呼吸動畫不運作？
**A:** 點擊「開始」按鈕後才會啟動動畫。確保 JavaScript 沒有被阻擋。

### Q5: 如何修改顏色？
**A:** 編輯 styles.css 中的 `:root` CSS 變數。

## 🌟 使用技巧

### Tip 1: 快速切換頁面
在任何頁面按 `Ctrl + U`（或 `Cmd + U`）查看源碼，可快速找到其他頁面連結。

### Tip 2: 截圖記錄
使用瀏覽器開發者工具（F12）的裝置模擬功能，精確截圖 iPhone 尺寸。

### Tip 3: 動畫調試
在 styles.css 中調整 `animation-duration` 可加快或減慢動畫速度。

### Tip 4: 色彩測試
使用瀏覽器擴充功能（如 ColorZilla）快速提取任意元素的顏色值。
